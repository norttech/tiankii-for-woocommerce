Tiankii Payment Gateway is a WooCommerce payment gateway that allows you to accept Lightning payments.

== Description ==

Tiankii Payment Gateway for WooCommerce enables your ecommerce store to accept Lightning payments in one unified Checkout experience.
This plugin integrates seamlessly with your WooCommerce Checkout, providing a fast, secure, and low-cost payment option for your customers.

**Features:**
- Accept Bitcoin, Lightning and Liquid payments directly to your wallet.
- Easy setup and configuration within WooCommerce settings.
- QR code generation for easy payments.
- Transaction history

**Why Choose Tiankii?**
- Lower transaction fees compared to traditional payment methods.
- Enhanced security and privacy for your customers.
- No intermediaries or third parties, receive payments directly.

== Installation ==

1. Navigate to _Plugins > Add New Plugin_, then search for 'tiankii' in the
   official WordPress directory.
2. Click 'Install Now', then once installed, click to Activate the plugin.
3. Navigate to _WooCommerce > Settings > Payments_.
4. Enable the Tiankii plugin, then click on 'Manage' to configure the payment
   gateway settings.
